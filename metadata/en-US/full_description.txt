Difuse Phone is a secure SIP client offering free audio/video calls and text messaging. Stay reachable at any time, even if the app is closed, with a WiFi or mobile data internet connection.

Key features:
* high definition audio and video calls
* audio conference calls with multiple participants
* pictures and file sharing
* reachable at any time via push notifications
* secure communication with encryption options
* compliant with a large number of SIP-compatible VoIP service providers

Difuse Phone is based on the open source Linphone project and is available on the main mobile and desktop operating systems. For more information, visit www.difuse.io
